# diceware
A command line diceware password helper

As per the instructions on the original diceware website, you will need an actual, physical die to use this method.
You first enter the number of words you would like in your password, then enter dice rolls one at a time. At the end, it gives you the option of generating a different password based on the numbers you entered (in case you find the one it generated unmemorable or offensive).
